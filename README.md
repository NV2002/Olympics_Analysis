# Olympics Data Analysis & Visualization (1896–2024)
This project visualizes Olympic trends from year 1896 to 2024 using an interactive D3.js heatmap. It highlights the top 10 Olympic sports by total medal count across all years.
The heatmap is fully interactive: hovering over a cell displays the sport, medal type, year, medal count, and indicates whether the cell represents the highest or lowest medal count in its category, with color-coded medals for quick visual insights.

## Features
- Interactive heatmap
- Useful details on hover (year, medal type, count, etc)
- Color-coded medal types

## Tech Stack
- HTML, CSS, D3.js
- Python (data preprocessing)
